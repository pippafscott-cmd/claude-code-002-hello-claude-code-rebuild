# Hello Claude Code — Lab Template

This is the starter template for the **Hello Claude Code** lab from the Claude AI Coding Bootcamp.

## What you'll build

A personalised "How well do you know me?" quiz that friends and family can take. You'll use Claude Code to add your own questions, change the look and feel, and make it entirely yours.

## Getting started

1. Clone this repository to your computer
2. Open the folder in the Claude Code Desktop App
3. Open `index.html` in your browser to see the starter quiz
4. Open `questions.js` — this is the only file you need to edit

## Your task

Follow the lab instructions to ask Claude to:

- Replace the two example questions with your own
- Add more questions until you have at least 8–10
- Personalise the results messages to match your personality
- Change the colours, fonts, or layout to make it look like yours

## File guide

| File | What it does |
|---|---|
| `questions.js` | **Your file** — questions, answers, name, and messages |
| `index.html` | The page structure — you can ask Claude to modify this too |
| `style.css` | All the styling — colours, fonts, layout |
| `quiz.js` | The quiz engine — no need to touch this |
